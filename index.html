Simpel

Ya web ini sangat simpel, kenapa? ada yang salah? apakah saya harus menambahkan banyak gambar dan tombol untuk sebuah web?

Persetan dengan navbar, slide, dll. Web ini tidak memaksa user untuk menload gambar atau file besar di dalamnya web ini sangat sempurna, mengapa?

  • Sangat ringan dan memuat dengan cepat
  • Cocok di semua layar jelek Anda
  • Terlihat sama di semua browser jelek Anda
  • Bajingan bisa diakses oleh setiap bajingan yang mengunjungi situs
  • Kontennya mudah dibaca dan bisa menyampaikan maksud (kalau Anda punya foto hipster yang sedang minum kopi, bukan hanya 5MB)

Estetik

Persetan dengan estetik, mengapa saya harus menggunakan banyak gambar dan tombol diawal baris hanya untuk menyampaikan pesan "HAI" dan mengapa saya harus menambahkan perubahan warna dan ukuran ketika kursor anda menyentuk tombol sialan yang tak berguna fungsiya. Itu hanya menghabiskan waktuku saja.
